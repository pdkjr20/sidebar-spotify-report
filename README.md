# The Side Bar Spotify Report

A data analysis of the Side Bar's Spotify playlist history — 104 playlists, 10,259 track adds, and nearly a decade of Friday nights.

## What's in here

- `index.html` — the full report (open in any browser)
- `assets/` — folder for images, data exports, or future additions

## How to update

1. Edit `index.html` (or have Claude update it)
2. Save the file
3. `git add .`
4. `git commit -m "describe what changed"`
5. `git push`

Netlify automatically redeploys within ~30 seconds of a push.
